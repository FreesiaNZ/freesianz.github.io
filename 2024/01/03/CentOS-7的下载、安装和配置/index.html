CentOS-7 的下载、安装和配置

发表于 2024-01-03 更新于 2024-03-16 分类于 Download & Install & Configure 本文字数： 836 阅读时长 ≈ 3 分钟

映像下载

首先进入 CentOS 的镜像下载站，然后下载标准版。

下载完毕，记住 iso 文件所在位置。

虚拟机创建

打开 VMware Workstation，然后点击 创建虚拟机 。

选择 自定义安装 ，然后点击 下一步 。

默认，然后点击 下一步 。

选择 稍候安装操作系统 ，然后点击 下一步 。

选择 Linux(L) ，选择 CentOS 7 64位 ，然后点击 下一步 。

设置 虚拟机名称 ，选择 位置 ，然后点击 下一步 。

选择 指定处理器数量 ，数量越多，性能越好，根据需要分配。然后点击 下一步 。

根据需要 分配内存 ，然后点击 下一步 。

网络连接选择 NAT，然后点击 下一步 。

默认，然后点击 下一步 。

默认，然后点击 下一步 。

选择 创建新虚拟硬盘 ，然后点击 下一步 。

指定 磁盘容量 ，然后点击 下一步 。

指定 磁盘文件存储 ，然后点击 下一步 。

点击 完成 。

安装和配置

点击 编辑虚拟机设置 。

点击 硬件 下的 CD/DVD ，选择 使用ISO映像文件 。

点击 开启此虚拟机 ，选择 Install CentOS 7，然后 ENTER 。

语言

选择语言为 中文 ，然后点击 继续 。

软件安装

点击 软件安装 。

选择 GNOME 桌面，附加选项勾选 GNOME 应用程序 , 传统 X Windows 系统的兼容性 和 开发工具 ，然后点击 完成 。

安装位置

点击 安装位置 。

选择 我要配置分区 ，然后点击 完成 。

设置 /boot ，点击 + 添加挂载点，挂载点选择 /boot ，期待容量设为 1024 。

设置 /swap ，点击 + 添加挂载点，挂载点选择 /swap ，期待容量设为 2048 。

设置 / ，点击 + 添加挂载点，挂载点选择 / ，期待容量设为 17408 。

点击 完成 ，点击 接收更改 。

网络和主机名

点击 网络和主机名 。

打开 以太网，设置 主机名 ，点击 应用 。

点击 开始安装 。

ROOT 密码

点击 ROOT 密码 。

设置 Root 密码 以及 确认 ，点击 完成 。

创建用户

点击 创建用户 。

设置 用户名 以及 密码 ，点击 完成 。

安装重启

熬过漫长的等待。

安装完成后，点击 重启 。

接收许可

点击 LICENSE INFORMATION 。

接受许可，点击 完成 后，点击 完成配置 。

登录账户

点击 未列出 ，使用 root 登录。

输入 用户名 和 密码 完成登录。

欢迎

出现桌面，欢迎、输入页选择 汉语 。

隐私页 关闭 位置服务。

跳过在线帐号页。

至此，CentOS-7 下载、安装和配置完毕！
