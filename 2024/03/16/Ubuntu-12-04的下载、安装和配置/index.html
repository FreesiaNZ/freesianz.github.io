Ubuntu-12.04 的下载、安装和配置

发表于 2024-03-16 更新于 2024-03-17 分类于 Download & Install & Configure 阅读次数： 本文字数： 519 阅读时长 ≈ 2 分钟

映像下载

首先进入 Ubuntu 的镜像下载站，然后下载桌面版。

下载完毕，记住 iso 文件所在位置。

虚拟机创建

打开 VMware Workstation，然后点击 创建虚拟机 。

选择 自定义安装 ，然后点击 下一步 。

默认，然后点击 下一步 。

选择 稍候安装操作系统 ，然后点击 下一步 。

选择 Linux(L) ，选择 Ubuntu ，然后点击 下一步 。

设置 虚拟机名称 ，选择 位置 ，然后点击 下一步 。

选择 指定处理器数量 ，数量越多，性能越好，根据需要分配。然后点击 下一步 。

根据需要 分配内存 ，然后点击 下一步 。

网络连接选择 NAT，然后点击 下一步 。

默认，然后点击 下一步 。

默认，然后点击 下一步 。

选择 创建新虚拟硬盘 ，然后点击 下一步 。

指定 磁盘容量 ，然后点击 下一步 。

指定 磁盘文件存储 ，然后点击 下一步 。

点击 完成 。

安装和配置

点击 编辑虚拟机设置 。

点击 硬件 下的 CD/DVD ，选择 使用ISO映像文件 。

点击 开启此虚拟机 。

欢迎

选择 中文 ，点击 安装 Ubuntu 。

准备安装 Ubuntu

不选择，点击 继续 。

安装类型

选择 清除整个磁盘并且安装 Ubuntu ，点击 继续 。

点击 现在安装 。

时区

选择 shanghai ，点击 继续 。

键盘布局

选择 汉语 ，点击 继续 。

用户名与密码

输入 用户名 和 密码 ，可勾选 自动登录 ，点击 继续 。

等待安装

等待安装完毕，点击 现在重启 。

安装成功
