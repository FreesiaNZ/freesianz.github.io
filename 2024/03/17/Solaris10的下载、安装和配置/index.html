Solaris 10 的下载、安装和配置

发表于 2024-03-17 分类于 Download & Install & Configure 本文字数： 987 阅读时长 ≈ 3 分钟

映像下载

使用的是 在 VirtualBox 上安装 Solaris 10 全教程（包括下载） 提供的资源。

下载完毕，记住 iso 文件所在位置。

虚拟机创建

打开 VMware Workstation，然后点击 创建虚拟机 。

选择 自定义安装 ，然后点击 下一步 。

默认，然后点击 下一步 。

选择 稍候安装操作系统 ，然后点击 下一步 。

选择 Linux(L) ，选择 Solaris 10 ，然后点击 下一步 。

设置 虚拟机名称 ，选择 位置 ，然后点击 下一步 。

选择 指定处理器数量 ，数量越多，性能越好，根据需要分配。然后点击 下一步 。

根据需要 分配内存 ，然后点击 下一步 。

网络连接选择 NAT，然后点击 下一步 。

默认，然后点击 下一步 。

默认，然后点击 下一步 。

选择 创建新虚拟硬盘 ，然后点击 下一步 。

指定 磁盘容量 ，然后点击 下一步 。

指定 磁盘文件存储 ，然后点击 下一步 。

点击 完成 。

使用映像文件

点击 编辑虚拟机设置 。

点击 硬件 下的 CD/DVD ，选择 使用ISO映像文件 。

点击 开启此虚拟机 。

配置选项

配置类型

选择 1.Solaris Interactive(default) 。

键盘布局

选择 US-English 。

语言

选择 6.Simplified Chinese 。

网络连接

选择 联网的 ，点击 下一步。（注意：图文不符）

DHCP

默认 是 ，点击 下一步。

Kerberos

默认 否 ，点击 下一步。

名称服务

默认 无 ，点击 下一步。

NFSv4 域名

默认 使用系统派生的 NFSv4 域 ，点击 下一步。

时区

默认 地域性的洲/国家/地区 ，点击 下一步。

洲与国家

选择 亚洲-中国 ，点击 下一步。

日期和时间

默认即可，点击 下一步。

root 口令

输入超级用户（root）口令，点击 下一步。

请记住，稍后登录需要。

远程服务

默认 是 ，点击 下一步。

确认信息

确认信息正确后，点击 确认。

安装选项

欢迎

配置结束，开始安装，点击 下一步 。

安装选项

默认 是 ，点击 下一步 。

指定媒体

默认 CD/DVD ，点击 下一步 。

许可证

勾选 接收 ，点击 下一步 。

安装类型

选择 自定义安装 ，点击 下一步 。

软件语言

勾选 中文(zh) 和 中文(UTF-8) ，点击 下一步 。

系统语言

选择 中文(UFT-8) ，点击 下一步 。

附加产品

默认 无 ，点击 下一步 。

Solaris 软件组

默认即可，点击 下一步 。

磁盘

默认即可，点击 下一步 。

磁盘分区

默认即可，点击 下一步 。

分布文件系统

默认即可，点击 下一步 。

准备安装

确认信息正确后，点击 立即安装 。

等待安装。

最后弹出多次对话框，均点击 继续 。

登录系统

输入用户名

输入用户名为超级用户（root）。

输入口令并确认

输入口令为之前已经设置的 root 口令。

选择桌面

默认 Java 桌面系统（版本3） ，点击 确定 。

进入系统
